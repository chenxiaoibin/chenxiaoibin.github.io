Sublime Text2使用技巧

发表于 2015-11-20   |   分类于 Sublime   |  

安装Sublime Text2

  • 下载Sublime Text2
  • 按照提示完成安装Sublime Text2

安装Package Control

  • 访问Package Control

  • 打开Sublime Text2，按下快捷键Ctrl+` ，打开了console，复制粘贴进去下边这段代码

    import urllib2,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) ); by = urllib2.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); open( os.path.join( ipp, pf), 'wb' ).write(by) if dh == h else None; print('Error validating download (got %s instead of %s), please try manual install' % (dh, h) if dh != h else 'Please restart Sublime Text to finish installation')

    按下Enter键完成安装

  • 重启Sublime Text2，按下快捷键Ctrl+Shift+p（也许会出现弹框提示，按照提示，进行重启操作）

安装Emmet插件

  • 启Sublime Text2，按下快捷键Ctrl+Shift+p，输入pci，选择Package Contro:Install Package，按Enter键进入，输入Emmet，选择第一个按Enter进行该插件的安装
  • 稍等片刻，重启Sublime Text2，按快捷键Ctrl+n新建一个文档，按快捷键Ctrl+Shift+p，输入sshtml，选择第一项，设置该文档的编辑语法规则为html
  • 文档编辑区输入!，注意是英文感叹号，按Tab键，则会自动生成html文档的一些默认标签，至此，Emmet安装成功
  • Emmet的快捷键用法请点击：Emmet快捷键

快捷键大全

  • Ctrl+Shift+P：打开命令面板
  • Ctrl+P：搜索项目中的文件
  • Ctrl+G：跳转到第几行
  • Ctrl+Shift+V：粘贴并格式化
  • Ctrl+D：游标，选择单词，重复增加选择下一个相同的单词
  • Ctrl+L：选择行，重复可依次增加选择下一行
  • Ctrl+Shift+L：选择多行
  • Ctrl+Shift+Enter：在当前行前插入新行
  • Ctrl+X：删除当前行
  • Ctrl+M：跳转到对应括号
  • Ctrl+J：选择标签内容
  • Ctrl+H：替换
  • Ctrl+R：前往 method
  • Ctrl+Shift+M：选中当前括号内容，重复可选着括号本身
  • Ctrl+/：注释当前行
  • Ctrl+Shift+/：当前位置插入注释
  • Ctrl+Alt+/：块注释，并Focus到首行，写注释说明用的
  • Ctrl+Shift+A：选择当前标签前后，修改标签用的
  • Alt+F3：选择所有相同的词
  • Alt+.：闭合标签
  • 按Ctrl，依次点击或选取，可需要编辑的多个位置
  • 按Ctrl+Shift+上下键，可替换行

支持中文

  • Ctrl+Shift+p
  • 输入install package，稍等
  • 输入ConvertToUTF8，点击后稍等
  • 重复步骤1、2，输入GBK Encoding Support，点击后稍等即可
