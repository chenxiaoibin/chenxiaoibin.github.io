安装Sublime Text2

  • 下载Sublime Text2
  • 按照提示完成安装Sublime Text2

安装Package Control

  • 访问Package Control

  • 打开Sublime Text2，按下快捷键Ctrl+` ，打开了console，复制粘贴进去下边这段代码

    import urllib2,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) ); by = urllib2.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); open( os.path.join( ipp, pf), 'wb' ).write(by) if dh == h else None; print('Error validating download (got %s instead of %s), please try manual install' % (dh, h) if dh != h else 'Please restart Sublime Text to finish installation')

    按下Enter键完成安装

  • 重启Sublime Text2，按下快捷键Ctrl+Shift+p（也许会出现弹框提示，按照提示，进行重启操作）

二分查找算法

发表于 2015-11-18   |   分类于 算法   |  

二分查找（BinarySearch）算法是用来查找有序数组中特定元素的算法。此算法用途广泛，执行高效。其原理是：搜索过程从数组的中间元素开始，判断数组的中间元素与查找元素的大小，如果相等，则中间元素为查找元素，如果大于中间元素，则在大于中间元素的那半部分数组中进行查找（接下来此查找过程也是分成两部分进行查找，原理相同），如果小于中间元素，则在另一半数组中查找，如果没有，则跳出查找过程，并返回提示信息。

import java.util.Arrays;

//二分查找算法
public class BinarySearch {

    /**
     * 
     * @param key ：要查找的值
     * @param a          ：有序的整数数组
     * @return
     */
    public static int rank(int key, int[] a) {
        int lo = 0;//数组的索引的最小值
        int hi = a.length - 1;//数组索引的最大值
        //进入while循环
        while(lo <= hi) {
            //将当前数组一分为二
            int mid = lo + (hi - lo) / 2;
            //如果key值小于数组中间值，则在数组左半边进行查找，如果key值大于数组中间值，则在数组右半边进行查找，如果没有该值，则返回-1
            if(key < a[mid]) {
                hi = mid - 1;
            } else if(key > a[mid]) {
                lo = mid + 1;
            } else {
                return mid;
            }
        }
        return -1;
    }
    public static void main(String[] args) {
        //定义一个数组
        int[] whiteList = {10, 24, 54, 87, 1, 2, 3, 98, 34,21,20, -1};
        //进行升序排列
        Arrays.sort(whiteList);
        System.out.println("查找的数值的索引为： " + rank(20, whiteList));
    }
}

欧几里得算法

发表于 2015-11-17   |   分类于 算法   |  

欧几里得算法又称辗转相除法，用于求两个非负整数的最大公约数。

public class Test {

    //欧几里得算法
    public static int gcd(int p, int q) {
        if(q == 0) {
            return p;
        }
        int r = p % q;
        return gcd(q, r);
    }
    //测试
    public static void main(String[] args) {
        System.out.println("两个非负整数的最大公约数为：" + gcd(2, 4));
    }
}

win7下使用Git

发表于 2015-11-16   |   分类于 Git   |  

第一步：安装Git客户端

  • 下载Git的windows客户端
  • 安装Git客户端

第二步：建立本地仓库

  • 在本地磁盘根目录下新建一个文件夹，名字随便起，例如：projects
  • 进入文件夹，右键选择Git Bash Here
  • 输入git config --global user.name "Your Name"，其中Your Name改为自己想起的名字
  • 输入git config --global user.email "email@example.com"，其中email@example.com改为自己的邮箱地址
  • 输入git init，将当前目录交给git进行管理，这是文件夹中会多出一个.git文件夹来
  • 在当前文件夹中新建一个txt文档，如：readme.txt，使用notepad++或vi打开，里面随便输入一些内容，保存
  • 输入git add readme.txt，把文件添加到仓库
  • 输入git commit -m "add readme.txt"，把文件提交给仓库

第三步：Github上建立远程仓库

  • 访问Github官网，没账号的注册一个账号
  • 登录进入，点击+ New Repository手动创建一个仓库，仓库名字自己起，如：projects
  • 切换到刚才的命令输入窗口，输入git remote add origin git@github.com:mvbin/projects.git，关联远程仓库，其中mvbin应改为你自己的github名称
  • 输入git push -u origin master，将本地分支与远程仓库的master分支关联
  • 刷新github网站页面，你会看见readme.txt已经添加到远程仓库中
    注意，添加文件至远程仓库，要先git add .或git add filename，再git commit -m "xxx"，然后再git push origin master

  • 命令窗口输入touch .gitignore，创建该文件旨在忽略一些不必提交的文件或文件夹
